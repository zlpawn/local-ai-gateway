Claude 代理 (Claude Code)

通过官方 API 或兼容 OpenAI 格式的代理服务转发 Anthropic 模型请求。

🚀 如何连接此网关？

网关支持通过不同路径后缀区分客户端。请在终端设置对应的环境变量：

# 给 Claude Code 使用:
export ANTHROPIC_BASE_URL="http://127.0.0.1:8787/code"
# 给 Claude Desktop 使用:
export ANTHROPIC_BASE_URL="http://127.0.0.1:8787/desktop"

* 注：若使用 Claude Desktop 客户端，请务必从配置了上述环境变量的终端启动应用（例如执行 `open -a "Claude"`）。

Codex 代理 (IDE)

为 IDE 插件（如 Cursor、GitHub Copilot 等）提供下游代理支持。

🚀 如何连接此网关？

在 Cursor 等 IDE 的“OpenAI Compatible”提供商中配置以下信息：

Base URL: http://127.0.0.1:8787/codex/v1
API Key: sk-随便填 (网关会自动处理并在下游节点中替换)
