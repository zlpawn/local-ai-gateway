Codex 代理

为兼容 OpenAI 接口的独立 Codex 客户端提供下游代理支持。

🚀 如何连接此网关？

对于独立的 Codex 桌面端或命令行：保存本页配置后，网关会自动生成模型目录文件 ~/.codex/gateway-model-catalog.json（官方模型 + 下方 Codex 节点里的第三方模型）。把下面片段贴到 ~/.codex/config.toml，并确保放在第一个 [section] 之前：

model_provider = "custom"
model_catalog_json = "~/.codex/gateway-model-catalog.json"
openai_base_url = "http://127.0.0.1:8787/codex/v1"

[model_providers.custom]
name = "Local AI Gateway"
base_url = "http://127.0.0.1:8787/codex/v1"
wire_api = "responses"
requires_openai_auth = true
experimental_bearer_token = "dummy"

说明：改 Codex 节点/模型后点保存即可刷新目录文件，不必手动跑 npm run codex:catalog。若 Desktop 已打开，保存后重启或重载 Codex 才能看到新模型。

切换到 model_provider = "custom" 后，旧会话可能暂时不显示。可点右上角「迁移历史会话」：先预览数量，确认后再写入；会自动备份 ~/.codex，不影响登录态和插件。
